# RAG Chatbot + CSV Agent Project

## Overview
This repository contains a Retrieval-Augmented Generation (RAG) chatbot and an AI agent project demonstrating function calling and tool usage with language models.

---

## Project 1: RAG Chatbot

### Description
A chatbot that uses retrieval-augmented generation to provide accurate, context-aware answers by combining a language model with external data.

### File
- `student_rag_project_gemma1b.ipynb`

---

## Project 2: CSV Concat Agent (Extra Credit)

### Description
An AI agent that performs CSV file concatenation and explores how language models interact with functions.

- Implements two versions:
  - Parameterized function (harder for agent)
  - Hardcoded function (easier for agent)
- Tests how prompts and docstrings affect function calling

### File
- `extra_credit_simple_student.ipynb`

---

## Tech Stack
- Python  
- Jupyter Notebook  
- LlamaIndex  
- Ollama (llama3.2)  
- Pandas  
- Vector Database (FAISS/Chroma)  

---

## How to Run
1. Open Jupyter Notebook  
2. Run:
   - `student_rag_project_gemma1b.ipynb`
   - `extra_credit_simple_student.ipynb`

## Conclusion
This project demonstrates both retrieval-based chatbots and agent-based function execution, highlighting key techniques in building intelligent AI systems.
